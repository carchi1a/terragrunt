name: no-shell
type: boolean
description: Disable shell functions in scaffold templates when using catalog.
parent: catalog
---

Shell functions in scaffold templates are **enabled by default**. The `--no-shell` flag allows you to disable them for security reasons when scaffolding modules through the catalog.

## Configuration Precedence

The setting can be configured in multiple ways with the following precedence order:

1. CLI flag (`--no-shell`)
2. Catalog configuration (`enable_shell = false`)

## Example

```bash
terragrunt catalog --no-shell
```

This disables shell functions when scaffolding modules through the catalog interface.
