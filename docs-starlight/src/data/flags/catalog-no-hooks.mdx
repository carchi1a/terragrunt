name: no-hooks
type: boolean
description: Disable hooks in scaffold templates when using catalog.
parent: catalog
---

Hooks in scaffold templates are **enabled by default**. The `--no-hooks` flag allows you to disable them for security reasons when scaffolding modules through the catalog.

## Configuration Precedence

The setting can be configured in multiple ways with the following precedence order:

1. CLI flag (`--no-hooks`)
2. Catalog configuration (`enable_hooks = false`)

## Example

```bash
terragrunt catalog --no-hooks
```

This disables hooks when scaffolding modules through the catalog interface.
